What does this module do?

  • It blocks the user interface properly during processing and shows a loading animation, avoiding unintended user interactions and improving the user experience.

Why do I need this module?

  • In Odoo 18, some processes don’t fully block the UI, which can lead to users clicking or navigating away before the process finishes. This module solves that by providing proper UI blocking with a loader.

Does this module work with custom modules?

  • Yes! It’s designed to integrate smoothly with both Odoo’s core modules and custom modules.

Will this module slow down my Odoo?

  • No, the module is lightweight and only activates during relevant processes, so normal performance is unaffected.

Is it compatible with Odoo 17 Enterprise and Community editions?

  • Yes, it supports both editions.
