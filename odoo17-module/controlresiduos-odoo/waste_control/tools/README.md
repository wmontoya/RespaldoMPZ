## Formateador de Código

### Python

Tener instalado Python (versión 3.12.0 o superior). Puedes descargarlo
desde [aquí](https://www.python.org/downloads/).

### Cspell

Tener instalado Node.js. Puedes descargarlo desde [aquí](https://nodejs.org/).

Instalar Cspell globalmente y el idioma español:

```bash
npm install -g cspell
npm install -g @cspell/dict-es-es
```

Desinstalar:

run `npm uninstall -g cspell`
run `npm uninstall -g @cspell/dict-es-es`

### Para formatear

Abrir el archivo con los comandos [Formateador](COMANDOS-PROYECTO)
Usar la extensión Markdown Command Runner para ejecutar los comandos desde el archivo
