# Guía de estandares del proyecto

## Textos

Para cualquier texto que se muestre al usuario sigue la regla:

* Solo la inicial del texto lleva mayúscula, a execion de nombres de lugares o abreviaciones especiales

* Para todo texto en código python o xml solo se usara ingles

* Estándar de nombres de archivos generados: El texto de nombre de archivo en reportes se usa inicial mayúscula y separadores "_" (f'Comparative_Fuel_Cost_Report_{object.year}')

## Nombres de archivos de código

En general se usa el snake_case

Para vistas se agrega la terminación _view
Para reportes se agrega la terminación_report
Para los wizard de Odoo se agrega la terminación _wizard

## Estándar de archivos

Para los archivos xml seguir el orden en su categoria ya sean vistas, reportes o wizard
