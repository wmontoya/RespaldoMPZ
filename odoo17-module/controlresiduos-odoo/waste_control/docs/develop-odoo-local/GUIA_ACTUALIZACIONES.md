# Guía de como realizar actualizaciones

## Cambios en archivos python

- Se debe de detener el servicio de Odoo y volver a iniciar, esto porque se debe de volver a compilar los archivos python
- Ir a el modulo de gestion de residuos y ingresar actualizar este modulo

## Cambios archivos xml

- Estos archivos se puede actualizar sin detener el servicio
- Ir a el modulo de gestion de residuos y ingresar actualizar este modulo

## Para actualizar traducciones .po

- se necesita de la extension de Odoo debug
- No necesita detener el servicio de Odoo
- Ir al apartado de configuraciones
- Ir a gestion de idiomas
- Actualizar el idioma deseado
